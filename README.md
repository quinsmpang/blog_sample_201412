SantaShooter
===

See [http://labs.gree.jp/blog/2014/12/12921/](http://labs.gree.jp/blog/2014/12/12921/) for the usage

Setup
---

* Cocos2d-x 3.3 RC2
* Cocos Studio 2.0.5
* Windows: Microsoft Visual Studio Community 2013
* Mac: Xcode 6.1.1 (Mac, non-iOS)
* node.js v0.10.33

node.js proxy
---

    $ cd proxy/node_modules
    $ npm build ws
    $ cd ..
    $ node proxy.js
